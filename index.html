Forest Fire Detection Using Satellite Images and Deep Learning

In this project, I created a python script to scrape the web for satellite images of forests and wildfires. I then reformatted the images into arrays, and trained a convolutional neural network classifier to detect forest fires with high accuracy. Finally, I deployed the model into an interactive streamlit app where users can upload their own images and recieve the probability of a forest fire occuring in that image.
